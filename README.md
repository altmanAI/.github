# AltmanAI — `.github`

### Humanity leads. Intelligence follows.

This repository is the organization-level governance, documentation, and R&D infrastructure layer for **AltmanAI**, by Altman Family Group LLC. It defines the public GitHub standards, contributor expectations, and default community health files that apply across the AltmanAI product ecosystem (DailyPilot, Sentinel Elite, and future releases) unless a specific repository overrides them.

AltmanAI builds human-first AI systems designed to support clearer thinking, stronger decision-making, and practical daily execution — grounded in the proprietary **P.A.I.H.I. framework** (Proof, Alignment, Integrity, Humanity, Impact).

## What we build

- **DailyPilot** — a human-first productivity co-pilot for daily planning, reflection, and intentional action. Live at [www.altmanai.tech](https://www.altmanai.tech).
- **Sentinel Elite** — a security and access-vault platform, part of the AltmanAI product ecosystem.
- **P.A.I.H.I. Framework** — the standard that grounds every AltmanAI product insight in real evidence.
- **Proof-of-Impact Ledger** — an internal framework turning daily actions into an auditable record.
- Additional research-backed concepts in development across the ecosystem (DecisionForge, ReleaseShield, AltmanCore, and related infrastructure).

## Governance model

- **BDFL (Benevolent Dictator for Life):** Blake Hunter Altman — Founder & Chief Executive Officer, AltmanAI by Altman Family Group LLC.
- **Official AI Technology Partner:** AltmanAI Model 2.0 — for advisory, drafting, research, development support, documentation architecture, governance preparation, and human-first AI system refinement.

## Approval rules

All significant changes to this repository — governance language, public profile content, contributor standards, or security policy — require explicit human review and sign-off from Blake Hunter Altman before merge or publication. SuperAgent and other AI development assistants may prepare files, branches, commits, and pull request drafts, but may not merge into `main` or publish final changes without that explicit approval, confirmed with the phrase **"All Clear for Impact."**

## Contributor expectations

Contributors — human or AI-assisted — are expected to follow the documentation and quality standards in `CONTRIBUTING.md`, disclose AI-assisted work where applicable, and never include secrets, credentials, or private data in issues, pull requests, or commits.
